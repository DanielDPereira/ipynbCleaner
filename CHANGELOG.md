# Changelog

## 0.2.0

- Removed `nbformat` dependency completely; package is now zero-dependency.
- Added file size metrics (original and final) and file size reduction calculation.
- Improved CLI to display beautiful, formatted summary with a progress bar in English.
- Overhauled README.md documentation with professional instructions and PyPI badges.

## 0.1.0

- Added a packaged notebook cleaner with a reusable core.
- Added a command-line interface for cleaning .ipynb files.
- Added regression tests and PyPI-ready packaging metadata.
- Added CI workflow for tests and distribution validation.
